# FoodTok app architecture (mock)

## Цель
Собрать современный UI-концепт приложения в стиле TikTok для рецептов с полным набором основных экранов.

## Слои
- `model/`: доменные модели (`RecipeVideo`, `Creator`, `Category`, `InboxItem`).
- `data/`: источник данных (`RecipeRepository`, `FakeRecipeRepository`).
- `domain/`: use-case (`GetAppDataUseCase`) для сборки данных приложения.
- `ui/app/`: точка входа UI, навигация вкладок и общий Scaffold.
- `ui/screens/`: отдельные экраны (Feed / Discover / Create / Inbox / Profile).
- `ui/components/`: переиспользуемые UI-компоненты карточек рецептов.
- `ui/theme/`: палитра и тема приложения.

## Экраны (моки)
1. **Feed** — лента с крупными mock-видео карточками, метаданными рецепта и социальными метриками.
2. **Discover** — категории и плитка трендов.
3. **Create** — экран сценария публикации рецепта (камера, озвучка, AI-описание).
4. **Inbox** — список уведомлений/сообщений.
5. **Profile** — профиль автора и грид опубликованных рецептов.

## Навигация
Bottom bar с 5 точками входа: Лента, Поиск, Создать, Inbox, Профиль.
